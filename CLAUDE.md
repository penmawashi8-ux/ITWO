# YouTube Shorts 自動生成システム 仕様書

## 動画仕様
- 解像度: 1080x1920（9:16縦型）
- 長さ: 25〜30秒
- フレームレート: 30fps
- スライド枚数: 4枚

## スライド構成
| 枚数 | 内容 |
|------|------|
| 1枚目（表紙） | 用語名（大）+ 英語名（小）+ "30秒でわかる"バッジ |
| 2枚目（定義） | 1文で完結する定義 |
| 3枚目（活用例） | 具体的なユースケース1〜2個 |
| 4枚目（まとめ） | 覚えるべきポイント1行 |

## スライドデザイン仕様
- 背景色: #1a2744（ネイビー）
- アクセント色: #f0a500（ゴールド）
- テキスト色: #ffffff（白）
- フォント: Noto Sans JP Bold
- 表紙の用語名: 80px
- その他テキスト: 48px
- 余白: 80px（上下左右）

## スクリプト生成プロンプト
以下のJSON形式のみで出力。前置き・マークダウン不要。

{
  "term": "用語名（日本語）",
  "term_en": "English name",
  "definition": "30字以内の定義",
  "use_case": "活用例（40字以内）",
  "point": "覚えるポイント（20字以内）",
  "narration": "ナレーション全文（150字以内・です/ます調）"
}

## 用語リサーチ方針
- 対象: ITエンジニア・DX推進担当者が知るべき最新用語
- 優先度高: 直近3ヶ月以内に注目された用語
- 除外: used_terms.jsonに記載済みの用語
- 毎回1用語のみ返す
- JSON形式で {"term": "用語名"} のみ返す
